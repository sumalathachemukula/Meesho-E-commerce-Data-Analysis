# Meesho E-Commerce Data Analysis

# Project Overview
A comprehensive Power BI analysis focusing on revenue performance, cancellation trends, and order lifecycle for a 1-year period.

##  Dashboard Reports

### 1. Cancellation & Revenue Loss Analysis
This page focuses on identifying where the business is losing money due to cancelled orders across different states and categories.

### 2. Revenue Leakage & Order Performance
A deep dive into the ratio of delivered revenue versus losses, featuring an order status breakdown and delivery day metrics.
![Revenue Leakage](Screenshot%202026-04-15%20101551.png)

### 3. Sales & Performance Analysis
An overview of high-level KPIs including Total Revenue ($49M), performance by top cities, and monthly revenue trends.
![Sales Performance](Screenshot%202026-04-15%20101712.png)

##  Key Technical Skills
* **Data Visualization:** Creating intuitive, theme-consistent dashboards in Power BI.
* **Business Metrics:** Calculating Cancellation Rates, Return Rates, and Revenue Leakage.
* **Geospatial Analysis:** Mapping performance across major Indian cities like Chennai, Mumbai, and Hyderabad.
